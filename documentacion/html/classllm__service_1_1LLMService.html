Detailed Description

Clase estática que encapsula las operaciones con el motor de Inteligencia Artificial (LLM).

Clase encargada de abstraer el consumo de servicios de inferencia LLM.

Inicializa una conexión HTTP orientada a flujos (streaming) para procesar tokens bajo demanda.

Activa el parámetro stream=True en el cliente HTTP de requests. Genera un bucle no bloqueante de memoria que lee las líneas binarias entrantes del socket, transformándolas en un generador nativo de Python a través de la directiva yield.

Parameters
promptTexto plano con la instrucción o consulta enviada al modelo.
Returns
generator Emite de forma asíncrona fragmentos (tokens) de texto decodificados en UTF-8.
