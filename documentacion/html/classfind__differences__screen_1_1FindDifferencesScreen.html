def find_differences_screen.FindDifferencesScreen._handle_canvas_click (   self,
  pos 
)
private

Evalúa si un clic táctil colisiona con el radio de acierto de alguna diferencia latente.

Convierte las coordenadas globales a márgenes locales de lienzo e infla un rectángulo de tolerancia (HIT_MARGIN) para compensar la imprecisión en pantallas táctiles resistivas/capacitivas.

Parameters
posTupla (x, y) con el píxel de contacto absoluto en la ventana de juego.
