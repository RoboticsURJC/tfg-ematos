Detailed Description

Controlador central y núcleo lógico del sistema robótico.

Coordina de manera asíncrona y mediante hilos el AssistantEngine, el FaceDisplay, el SessionManager, la StateMachine y la interfaz gráfica de ventanas de PyQt5.

◆ __init__()

def app.core.robot_controller.RobotController.__init__ (   self)

Inicializa todos los subsistemas fundamentales del hardware y software del robot.

Resuelve de forma dinámica y absoluta la ruta hacia config/config.json, parsea las variables de red para el servidor de inferencia LLM y levanta las rutinas de los planificadores (schedulers) de alertas y sugerencias proactivas.

def app.core.robot_controller.RobotController.on_proactive_dismissed (   self)

Comunica al planificador que la tarjeta o modal de sugerencia proactiva ha sido cerrada.

Debe invocarse obligatoriamente desde el hilo de la UI cuando el usuario interactúe con los botones de acción ('Aceptar' o 'Ahora no') para restaurar el display gráfico facial.

def app.core.robot_controller.RobotController.shutdown (   self)

Transiciona el estado a SHUTDOWN, apaga los bucles de escucha del motor de voz, cancela los hilos de los schedulers y clausura de forma nativa la UI de PyQt5.
