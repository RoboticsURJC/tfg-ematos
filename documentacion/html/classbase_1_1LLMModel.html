Detailed Description

Clase base abstracta para la gestión y abstracción de modelos de lenguaje (LLM).

Define la interfaz común que deben implementar obligatoriamente las subclases (ej: GroqClient, LocalLlama, OpenAIClient) para la generación de texto y evaluación de rendimiento.

Ejecuta una prueba de rendimiento (benchmark) controlada sobre el método de generación.

Envuelve la llamada al método de inferencia generate(), inyecta trazas de depuración en la consola usando metatablas de la clase activa y retorna los datos para auditorías de latencia.

Parameters
promptTexto de entrada o benchmark estándar para evaluar el comportamiento del modelo.
Returns
tuple Una tupla (response, extra) idéntica a la respuesta del método de generación implementado.

Genera una respuesta de texto a partir de una instrucción o prompt estructurado.

Este es un método abstracto puro. Las subclases derivadas deben sobrescribir este método para comunicarse con sus respectivos backends o APIs de inferencia.

Parameters
promptCadena de texto limpia que contiene las directivas e historial de usuario.
Returns
tuple Una tupla compuesta por (response, extra) donde:
  • response (str): El texto plano generado por el modelo.
  • extra (dict): Metadatos de la inferencia (tokens por segundo, tiempo de ejecución, etc.).
Exceptions
NotImplementedErrorLanzada si se invoca el método directamente desde la clase base sin haber sido sobrescrito.
