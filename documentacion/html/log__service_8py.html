Detailed Description

Servicio de almacenamiento y gestión de registros de eventos (Logs) en memoria.

Proporciona funciones con seguridad de hilos (Thread-safe) para añadir, consultar y vaciar los registros de auditoría de los distintos componentes del sistema.
