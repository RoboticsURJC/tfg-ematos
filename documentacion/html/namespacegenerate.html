Valida mediante proyección lineal si una palabra puede ser sembrada en una coordenada específica.

Evalúa de forma preventiva dos criterios críticos:

  1. Desbordamiento de límites físicos del tablero (Out of Bounds).
  2. Colisión de caracteres (evita sobreescrituras a menos que la celda coincida exactamente con la misma letra).
Parameters
matriceReferencia directa a la matriz bidimensional (list de lists).
wordCadena de caracteres del término a validar.
indexTupla (x, y) con la coordenada propuesta para la primera letra del término.
fit_dirTupla (dx, dy) obtenida del diccionario de posibilidades que rige el crecimiento lineal.
Returns
bool True si la región es apta y segura para la siembra de caracteres; False en caso contrario.

◆ generate_board()

def generate.generate_board ( )

Genera procedimentalmente un nuevo mapa completo de sopa de letras bajo criterios de optimización espacial.

Algoritmo de escape por recursividad adaptativa: Si el factor de empaquetamiento espacial colapsa y una palabra no logra situarse tras superar el umbral de seguridad MAX_ATTEMPTS, el motor captura el bloqueo, reduce de forma dinámica la cuota de palabras deseadas en una unidad y reinicia el proceso de generación de forma recursiva para mitigar bucles infinitos de procesamiento en la CPU de la Raspberry Pi. Tras la siembra limpia, rellena el ruido residual con caracteres pseudoaleatorios extraídos del alfabeto anglosajón.

Returns
bool True cuando el tablero logra estructurarse de forma integral y queda listo para el render.

◆ generate_word_list()

def generate.generate_word_list ( )

Carga y normaliza el banco de términos lingüísticos desde el sistema de archivos del dispositivo.

Abre el descriptor en modo lectura segura encoding="utf-8". Ante un fallo de entrada/salida (I/O), captura la excepción e inyecta un pool alternativo de palabras hardcodeadas para garantizar la resiliencia del software.

Returns
list Colección de cadenas de texto en mayúsculas listas para su inserción estructural.

Modifica físicamente las celdas de la matriz e inyecta los caracteres de la palabra validada.

Almacena de forma indexada en el diccionario global word_positions la secuencia exacta de tuplas de coordenadas discretas ocupadas, permitiendo que la interfaz táctil realice comprobaciones instantáneas.

Parameters
matriceReferencia directa a la matriz bidimensional activa de caracteres.
wordCadena de caracteres del término a inyectar.
indexTupla (x, y) inicial de siembra.
fit_dirTupla (dx, dy) que gobierna la dirección de escritura en la memoria matricial.
