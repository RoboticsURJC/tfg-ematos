Procesa una imagen para reconocer un rostro.

Envía la imagen capturada al servicio de reconocimiento facial para contrastarla con la base de datos de usuarios registrados.

Parameters
payloadObjeto RecognizeRequest que contiene la imagen en formato compatible (generalmente Base64 o ruta).

Registra a una nueva persona en el sistema.

Almacena un nuevo recuerdo o dato textual para un usuario.
