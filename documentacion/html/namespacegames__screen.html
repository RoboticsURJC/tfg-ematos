str games_screen._write_svg ( str  svg_text,
str  color 
)
private

Inyecta una clave de color hexadecimal en el XML del SVG y lo escribe en el directorio temporal del sistema.

Optimización de renderizado: Evita cuellos de botella e incompatibilidades de parseo en caliente sobre hilos de ejecución de la Raspberry Pi aislando físicamente el recurso en disco.

Parameters
svg_textCadena con la estructura XML nativa del SVG que posee la directiva comodín 'ICONCOLOR'.
colorCadena de texto hexadecimal (Ej: '#ffffff') con el color final de trazo.
Returns
str Ruta absoluta física hacia la entidad temporal generada (ej: /tmp/tmpXXXXXX.svg).
