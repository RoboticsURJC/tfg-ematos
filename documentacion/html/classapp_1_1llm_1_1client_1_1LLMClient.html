Inicializa los parámetros de red y configura las directrices del modelo.

Limpia la URL base eliminando barras diagonales residuales y anexa el endpoint estricto /generate requerido por la arquitectura del servidor de inferencia.

Realiza una consulta síncrona al modelo de lenguaje y devuelve la respuesta consolidada.

Concatena las directrices del system_prompt con la entrada actual del usuario, valida el código de estado HTTP (r.raise_for_status) y extrae el texto plano generado.

Parameters

Returns
str Texto de respuesta generado de forma completa por la Inteligencia Artificial.

Inicia una petición de red por flujo continuo (Streaming) para recibir texto token a token.

Utiliza la directiva stream=True de la librería requests e itera sobre las líneas de bytes entrantes mediante un generador de Python (yield).

Parameters
promptTexto consolidado de entrada para la generación por flujo.
Returns
generator Devuelve un generador que emite líneas codificadas en UTF-8 conforme llegan del servidor.
