◆ _kill_all()

def browser_screen.BrowserScreen._kill_all (   self)
private

Detiene y purga de forma atómica todos los procesos e instancias zombis del navegador y teclados auxiliares.

Emplea una política de terminación gradual benigna (terminate), escalando a interrupción forzada de kernel (kill) si el proceso bloquea los descriptores de archivos tras una ventana de 2 segundos.

◆ _launch_chromium()

def browser_screen.BrowserScreen._launch_chromium (   self,
  url 
)
private

Configura y levanta el subproceso independiente de Chromium optimizado para sistemas integrados.

Cierra de forma preventiva instancias zombis concurrentes y parametriza un array masivo de flags de aceleración de hardware, omisión de diálogos de error por cuelgues (--noerrdialogs) y perfiles de color estables para mitigar la fatiga gráfica en pantallas de Raspberry Pi.

Parameters
urlDirección web absoluta que cargará el navegador en su arranque inicial.
