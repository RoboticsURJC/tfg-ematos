Detailed Description

Inicialización del mezclador de audio de bajo nivel.

Componente contenedor intermedio que hospeda un bucle de ejecución (Game Loop) de Pygame en Qt.

◆ _blit_to_label()

def base_pygame_qt_screen.BasePygameQtScreen._blit_to_label (   self)
private

Transfiere los píxeles de la memoria de Pygame hacia el backend de dibujado de Qt.

Optimización crítica ARM: Recupera la matriz en 3D mediante surfarray.array3d, aplica una transposición de ejes ortogonales y fuerza la copia de manera contigua en memoria RAM mediante NumPy. Esto elude las corrupciones y cuelgues que provoca la rutina nativa image.tostring bajo arquitecturas Linux aarch64.

def base_pygame_qt_screen.BasePygameQtScreen._tick (   self)

Ejecuta una iteración unificada del ciclo del juego (Fase lógica + Fase gráfica).

Se ejecuta secuencialmente bajo el hilo principal de Qt. Sincroniza y vacía el vector de eventos de entrada al finalizar cada fotograma para impedir acumulaciones fantasmas de clicks táctiles.
