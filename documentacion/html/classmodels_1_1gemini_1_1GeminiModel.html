)

Genera texto a partir de un prompt utilizando el modelo ligero y rápido Gemini Flash.

Envuelve la llamada en un bucle tolerante de hasta 3 intentos. Si el servidor de Google devuelve un error HTTP 429 (Resource Exhausted / Rate Limit), el hilo de ejecución se congela de forma incremental antes de reintentar la inferencia.

Note
La fórmula de espera ante un error 429 aplica un retraso lineal/exponencial básico: espera = 10 * (intento_actual + 1). Esto se traduce en pausas sucesivas de 10, 20 y 30 segundos.
Parameters
promptTexto estructurado con el contexto e instrucciones para el modelo.
Returns
tuple Una tupla (response_text, extra_metadata) donde response_text es el string generado.
Return values
(None,dict)Retorna un diccionario con una bandera de error si se agotan todos los reintentos permitidos.
