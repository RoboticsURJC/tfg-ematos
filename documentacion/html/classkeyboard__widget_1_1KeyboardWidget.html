Detailed Description

Teclado software interactivo diseñado para alta legibilidad.

Proporciona una interfaz táctil de teclado con disposición QWERTY, soporte para mayúsculas y una matriz matemática fija de 10 columnas para garantizar una alineación consistente en diferentes resoluciones de pantalla.

@attr confirmed Señal emitida cuando el usuario presiona el botón 'Listo'.
