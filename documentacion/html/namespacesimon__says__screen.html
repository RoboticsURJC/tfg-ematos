Algoritmo iterativo de aproximación por convergencia para ajustar el tamaño de fuente.

Realiza una búsqueda lineal descendente garantizando que el texto renderizado quepa estrictamente dentro de los límites físicos (bounding box) horizontales y verticales.

Parameters
textCadena de caracteres a medir.
max_wAncho máximo disponible en píxeles.
max_hAlto máximo disponible en píxeles.
boldBandera lógica para habilitar el estilo de tipografía en negrita.
Returns
pygame.font.Font Instancia tipográfica optimizada para las dimensiones dadas.
