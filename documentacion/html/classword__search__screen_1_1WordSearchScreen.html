Algoritmo analítico de restricción ortogonal y diagonal para vectores de celdas.

Garantiza mediante normalización de pendientes unitarias recurrentes que las celdas arrastradas formen estrictamente una línea recta continua (horizontal, vertical o diagonal).

Returns
