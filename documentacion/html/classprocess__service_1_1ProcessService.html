def process_service.ProcessService.is_running (   cls,
str  service 
)

Evalúa si el subproceso asociado a un servicio sigue activo.

Utiliza el método .poll() nativo de Python; si retorna None, significa que el proceso sigue en ejecución y no ha terminado.

Parameters
serviceIdentificador del servicio a monitorizar.
Returns
bool Verdadero (True) si está corriendo, Falso (False) en caso contrario.

Levanta un microservicio en segundo plano de manera asíncrona.

Verifica si el servicio existe y si no está corriendo ya. Si pasa los filtros, inicializa una nueva sesión de proceso (start_new_session=True) para encapsular las llamadas de terminal.

def process_service.ProcessService.stop (   cls,
str  service 
)

Detiene de forma segura el microservicio indicado liberando sus recursos.

Obtiene el ID de grupo de proceso (PGID) para enviar una señal de terminación suave (SIGTERM) al árbol de procesos de Uvicorn. Si no responde tras un segundo de espera, se fuerza su cierre con SIGKILL.

Parameters
serviceIdentificador del servicio a detener ('llm' o 'recognition').
