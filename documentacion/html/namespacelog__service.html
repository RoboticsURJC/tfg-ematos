def log_service.add_log ( str  service,
str  message 
)

Añade una nueva entrada de registro para un servicio específico.

Crea una estructura con la hora exacta y el mensaje recibido, la anexa a la lista del servicio correspondiente y trunca el historial si supera el límite establecido en MAX_LOGS. Este método está protegido de forma concurrente mediante un cerrojo.

Parameters
serviceNombre de la categoría o componente (ej: 'llm', 'recognition', 'system').
messageTexto descriptivo del evento o error que se desea registrar.
