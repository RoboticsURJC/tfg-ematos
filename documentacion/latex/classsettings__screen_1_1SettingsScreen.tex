\doxysubsection{Detailed Description}
Pantalla principal de ajustes. 

Interfaz principal de ajustes.

Conecta señales asíncronas para actualizar la UI desde hilos de hardware.

Utiliza señales personalizadas para actualizar la UI desde hilos secundarios, evitando bloqueos durante procesos lentos como el escaneo Bluetooth.
