Evalúa si el subproceso asociado a un servicio sigue activo. 

Utiliza el método {\ttfamily .poll()} nativo de Python; si retorna {\ttfamily None}, significa que el proceso sigue en ejecución y no ha terminado.

Detiene de forma segura el microservicio indicado liberando sus recursos. 

Obtiene el ID de grupo de proceso (PGID) para enviar una señal de terminación suave ({\ttfamily SIGTERM}) al árbol de procesos de Uvicorn. Si no responde tras un segundo de espera, se fuerza su cierre con {\ttfamily SIGKILL}.
