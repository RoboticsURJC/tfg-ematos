{\footnotesize\ttfamily def models.\+gemini.\+Gemini\+Model.\+generate (\begin{DoxyParamCaption}\item[{}]{self,  }\item[{}]{prompt }\end{DoxyParamCaption})}



Genera texto a partir de un prompt utilizando el modelo ligero y rápido Gemini Flash. 

Envuelve la llamada en un bucle tolerante de hasta 3 intentos. Si el servidor de Google devuelve un error HTTP 429 (Resource Exhausted / Rate Limit), el hilo de ejecución se congela de forma incremental antes de reintentar la inferencia.

\begin{DoxyNote}{Note}
La fórmula de espera ante un error 429 aplica un retraso lineal/exponencial básico\+: {\ttfamily espera = 10 $\ast$ (intento\+\_\+actual + 1)}. Esto se traduce en pausas sucesivas de 10, 20 y 30 segundos.
\end{DoxyNote}
