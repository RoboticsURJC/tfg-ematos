\doxysubsection{Detailed Description}
Clase base abstracta para la gestión y abstracción de modelos de lenguaje (LLM). 

Define la interfaz común que deben implementar obligatoriamente las subclases (ej\+: Groq\+Client, Local\+Llama, Open\+AIClient) para la generación de texto y evaluación de rendimiento.
