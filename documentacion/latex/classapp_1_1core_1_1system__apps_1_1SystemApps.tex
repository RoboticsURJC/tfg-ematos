Clase estática encargada del enrutamiento y ejecución segura de aplicaciones. 

Incorpora un mecanismo electrónico/lógico anti-\/rebote (de-\/bounce) para pantallas táctiles.

{\footnotesize\ttfamily def app.\+core.\+system\+\_\+apps.\+System\+Apps.\+launch (\begin{DoxyParamCaption}\item[{}]{cls,  }\item[{}]{app\+\_\+id }\end{DoxyParamCaption})}



Ejecuta o enruta una aplicación del sistema aplicando filtros de seguridad táctil. 

Evalúa la diferencia de tiempo desde la última pulsación para interceptar dobles clics accidentales. Si el ID de la aplicación está marcado como {\ttfamily internal\+:}, delega el control retornando el prefijo; si es una lista de comandos del sistema, barre el PATH del sistema operativo usando {\ttfamily shutil.\+which} para lanzar el binario disponible en segundo plano de forma no bloqueante.


\begin{DoxyParams}{Parameters}

\end{DoxyParams}

bool Retorna True si la aplicación externa se ejecutó con éxito, o False ante errores o app no registrada.
