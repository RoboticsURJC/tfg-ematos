\doxysubsection{Detailed Description}
Módulo wrapper y cliente de consumo para los endpoints de la API del servidor LLM. 

Simplifica el acceso a las funciones de inferencia de texto, empaquetando las llamadas HTTP POST síncronas estándar y el procesamiento de flujos orientados a tokens (streaming).
