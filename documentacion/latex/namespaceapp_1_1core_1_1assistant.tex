Orquesta el flujo principal (Pipeline) para procesar un mensaje del usuario. 

Ejecuta la toma de decisiones del robot siguiendo cuatro etapas claras\+:
\begin{DoxyEnumerate}
\item Invoca el módulo analítico para detectar la intención subyacente del texto.
\item Bifurca el flujo ejecutando una herramienta local rápida (hora, clima) o delegando al LLM.
\item Anexa el par de interacción pregunta/respuesta a la estructura de datos en memoria.
\item Sincroniza y escribe la actualización en el almacenamiento persistente en disco.
\end{DoxyEnumerate}
