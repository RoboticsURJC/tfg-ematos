Endpoint HTTP POST\+: Orquesta la inferencia del modelo y audita sus tiempos. 

Valida la existencia del proveedor en el diccionario {\ttfamily models}, intercepta strings vacíos para economizar recursos de red y captura excepciones a nivel de socket mediante {\ttfamily traceback} para impedir el colapso del hilo de ejecución asíncrono (Fail-\/\+Safe).
