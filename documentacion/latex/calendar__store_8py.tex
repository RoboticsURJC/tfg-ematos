\doxysubsection{Detailed Description}
Gestor de persistencia local para el almacenamiento de eventos de la agenda. 

Implementa una capa de abstracción sobre un archivo plano en formato JSON para guardar, listar y eliminar eventos cronológicos de forma síncrona, manteniendo una copia activa en memoria RAM.
