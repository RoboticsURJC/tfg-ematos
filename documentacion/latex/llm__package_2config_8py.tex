\doxysubsection{Detailed Description}
Módulo central de configuración y carga de variables de entorno del sistema. 

Consume la librería {\ttfamily python-\/dotenv} para deserializar el archivo oculto {\ttfamily .env} e inyectar de forma segura las credenciales y llaves privadas en las variables del sistema operativo.
