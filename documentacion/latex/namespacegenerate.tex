Genera procedimentalmente un nuevo mapa completo de sopa de letras bajo criterios de optimización espacial. 

{\bfseries{Algoritmo de escape por recursividad adaptativa\+:}} Si el factor de empaquetamiento espacial colapsa y una palabra no logra situarse tras superar el umbral de seguridad {\ttfamily MAX\+\_\+\+ATTEMPTS}, el motor captura el bloqueo, reduce de forma dinámica la cuota de palabras deseadas en una unidad y reinicia el proceso de generación de forma recursiva para mitigar bucles infinitos de procesamiento en la CPU de la Raspberry Pi. Tras la siembra limpia, rellena el ruido residual con caracteres pseudoaleatorios extraídos del alfabeto anglosajón.

\begin{DoxyReturn}{Returns}
bool True cuando el tablero logra estructurarse de forma integral y queda listo para el render. 
\end{DoxyReturn}
