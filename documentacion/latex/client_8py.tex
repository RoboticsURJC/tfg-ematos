\doxysubsection{Detailed Description}
Cliente LLM de red y conector para la API de inferencia del modelo de lenguaje (LLM). 

Gestiona las peticiones HTTP síncronas y por flujo (streaming) hacia el backend de inteligencia artificial configurado, encapsulando las directivas del sistema y el control de errores.
