\doxysubsection{Detailed Description}
Manejador de logs especializado en la persistencia y centralización remota via HTTP POST. 

Intercepta cualquier evento emitido en los niveles configurados (ej\+: DEBUG, INFO) y lo despacha síncronamente por red hacia la base de datos de telemetría.

Procesa, formatea y transmite el registro de log de forma activa por red. 

Extrae las propiedades textuales del evento utilizando los formateadores configurados en el objeto padre, construye un diccionario JSON con la clave \textquotesingle{}text\textquotesingle{} y ejecuta una petición HTTP POST.

\begin{DoxyNote}{Note}
El tiempo de espera (timeout) está limitado estrictamente a 1 segundo. Si el Dashboard está saturado, desconectado o en una IP errónea en la red local de la Raspberry Pi, el hilo no se quedará colgado de forma indefinida y la ejecución del robot continuará con normalidad.
\end{DoxyNote}
