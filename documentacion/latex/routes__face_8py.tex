\doxysubsection{Detailed Description}
Enrutador de API para el sistema de reconocimiento facial y gestión de memorias. 

Ofrece endpoints para registrar rostros, reconocer personas y gestionar la base de datos de conocimientos/memorias asociadas a los usuarios.
