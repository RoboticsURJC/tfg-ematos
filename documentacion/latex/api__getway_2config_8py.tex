\doxysubsection{Detailed Description}
Archivo de configuración global del sistema y definición de constantes. 

Inicializa las rutas del sistema de archivos (directorios de logs y archivos estáticos), define los puntos de enlace de los microservicios y configura los parámetros de red del dashboard.
